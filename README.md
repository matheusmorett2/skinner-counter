
# Skinner Counter


### Motivation

This project was made for help psychologist count actions of the rat based on ** B. F. Skinner Methodology **
  

### Dependencies

- react

- react-csv

### Instructions:

1. Run: `yarn && yarn start` 

3. Go to http://0.0.0.0:3000/
